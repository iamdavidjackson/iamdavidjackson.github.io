# iamdavidjackson.github.io

Personal site for [iamdavidjackson.com](https://iamdavidjackson.com), built with Jekyll and deployed with GitHub Pages.

## Local development

1. Install Ruby and Bundler.
2. Run `bundle install`.
3. Run `bundle exec jekyll serve`.
4. Open `http://127.0.0.1:4000`.

## Notes

- The site is configured to work both locally and on GitHub Pages.
- Pagination is powered by `jekyll-paginate`, which GitHub Pages supports.
- Production URLs are generated from `https://iamdavidjackson.com`.
